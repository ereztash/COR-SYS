## סינתזה — איך הפער נסגר

1. **מסמך זה (docs/synthesis-html-app.md)** — ממפה פער ומתעד.
2. **דף "זהות עסקית" באפליקציה (`/about`)** — מציג את ארבע הלשוניות MECE (זהות, הבעיה, הפתרון, מסחור) בתוך האפליקציה, עם תוכן מעובד מה-HTML.
3. **תיקון ניסוח:** ב-HTML מופיע "עובד סוציאלי קליני" — בפרופיל (identity.md) זה תואר מוסדר ואסור. בדף באפליקציה משתמשים בניסוח: "עבודה סוציאלית", "מדעי ההתנהגות", "עובד סוציאלי טכנולוגי".
4. **ניווט:** קישור בסיידבר ל"זהות עסקית" (או "אודות") לדף `/about`.
