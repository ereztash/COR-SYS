# ICP ומכירות

---

## ICP (Ideal Customer Profile)

- **גודל:** 50–300 עובדים  
- **שלב:** Growth Stage — Series A עד C  
- **מצב:** עברו את Dunbar's Number (150+). רשתות לא-פורמליות לא נושאות את עומס התיאום. עדיין לא בנו מבנים היררכיים שעובדים.

### סקטורים עדיפים

| סקטור | מדוע |
|--------|------|
| Cybersecurity | רגולציה + ריבוי תקרית → decision latency גבוה |
| Fintech | תאימות + קצב שוק → אנטרופיה מבנית מהירה |
| Healthtech (AI) | ממשק אנושי-מערכת = החיכוך ש-COR-SYS פותר |
| B2B מורכב | תלות בין-מחלקתית = כשל handoff קלאסי |

### Champions (מי מגייס)

- **COO:** חש צוואר בקבוק; הצוות עובד קשה אבל לא מתקדם.
- **CFO:** רואה שעות ללא מתאם להכנסות; רוצה ROI.
- **CEO:** מגיע ל-Live Demo, רואה את ההפסד, מגייס לספרינט.
